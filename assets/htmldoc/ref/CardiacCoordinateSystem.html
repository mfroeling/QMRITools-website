CardiacCoordinateSystem

CardiacCoordinateSystem[mask, vox]
creates the cardiac coordinate system within the mask. output is a set of vectors {radvecn, norvecc, cirvec}, being the radial, normal and circular axes of each voxel respectivley.

DetailsDetails

  • The following options can be given:
  • ShowFitFalseShowFit is an option for CentralAxes. True shows the fit of the central axes.
