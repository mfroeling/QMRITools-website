DenoiseCSIdata

DenoiseCSIdata[spectra]
perfroms PCA denoising of the complex values spectra, data has to be 3D and the spectral dimensions is last, {x,y,z,spectra}.

DetailsDetails

  • The following options can be given:
  • PCAKernel5PCAKernel is an option of PCADeNoise. It sets the kernel size.
    PCANoiseSigmaCornersPCANoiseSigma is an option of DenoiseCSIdata and can be "Corners" or "Automatic".
