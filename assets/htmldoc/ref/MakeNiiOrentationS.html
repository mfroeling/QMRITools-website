MakeNiiOrentationS

MakeNiiOrentationS[off, vox]
maxes the srow values for nii header assuming not R and Q.

MakeNiiOrentationS[off, vox, R]
maxes the srow values for nii header using rotation R.

MakeNiiOrentationS[off, vox, R, Q]
maxes the srow values for nii header using rotation R and skew Q.
