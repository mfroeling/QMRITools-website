CoilWeightedReconCSI

CoilWeightedReconCSI[kspace, noise, head]
performs reconstuction of raw 3DCSI data. The input kspace, noise and head are obtained using ReadListData. The coil combination Methods can be "Roemer" or "WSVD".

DetailsDetails

  • The following options can be given:
  • HammingFilterFalseHammingFilter is an option for CoilWeightedReconCSI. If True it applies a spatial hamming filter to the data
    CoilSamples5CoilSamples is an option for CoilWeightedReconCSI and specifies how many fud samples are used to calcualte the coil sensitivity for Roemer reconstruction.
    MethodRoemerMethod is an option for various algorithm-intensive functions that specifies what internal methods they should use.
    NormalizeOutputSpectraTrueNormalizeOutputSpectra is an option for CoilWeightedReconCSI.
