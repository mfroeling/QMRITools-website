GetMaskMeans

GetMaskMeans[dat, mask, name]
calculates the mean, std, 5,50 and 95% CI form the given data for each of the given masks. Mask can be genereated by SplitSegmentations. name is a string that is added to the header.

DetailsDetails

  • The following options can be given:
  • MeanMethodSkewNormalDistMeanMethod is an option for GetMaskMeans. The option can be "NormalDist", "SkewNormalDist", or "Mean".
