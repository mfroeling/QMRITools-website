ListSpherePlot

ListSpherePlot[points]
plots 3D points as spheres

DetailsDetails

  • The following options can be given:
  • SphereSize2SphereSize is an option for GradientPlot and ListSpherePlor. Sets the size of the spheres thar represent the gradients.
    SphereColorAutomaticSphereColor ListSpherePlor. Default value is Automatic, If a color is given this color will be used for all spheres.
