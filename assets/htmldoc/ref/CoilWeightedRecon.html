CoilWeightedRecon

CoilWeightedRecon[kspace, noise, head]
performs reconstuction of raw MS2D MRI data. The input kspace, noise and head are obtained using ReadListData. The coil combination Methods can be "Roemer" or "RSS".

DetailsDetails

  • The following options can be given:
  • EchoShiftData0EchoShiftData is an option for CoilWeightedRecon
    CoilSamples2CoilSamples is an option for CoilWeightedReconCSI and specifies how many fud samples are used to calcualte the coil sensitivity for Roemer reconstruction.
    MethodRoemerEqualSignalMethod is an option for various algorithm-intensive functions that specifies what internal methods they should use.
