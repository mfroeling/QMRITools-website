PadToDimensions

PadToDimensions[data, dim]
pads the data to dimensions dim.

DetailsDetails

  • The following options can be given:
  • PadValue0.PadValue is an option for PadToDimensions. It specifies the value of the padding.
    PadDirectionCenterPadDirection is an option for PadToDimensions. It specifies the direction of padding, "Center", "Left" or "Right".
