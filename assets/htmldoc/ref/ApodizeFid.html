ApodizeFid

ApodizeFid[fid]
performs apodization on the fid. The apodization function is set with the option ApodizationFunction.

DetailsDetails

  • The following options can be given:
  • ApodizationFunctionHanningApodizationFunction is an options for ApodizeFid, ApodizeSpectra, ApodizePadFid, and ApodizePadSpectra. Values can be "Hanning", "Hamming", "Gaussian", "Lorentzian", and "Voigt".
