ExportTCK

ExportTCK[file,tracts,vox,dim]

exports the tracts to the given *.tck file and automatically generates the header.

ExportTCK[file,tracts,header]

exports the tracts to the given *.tck file. The header can be the header string or the header association as given by ImportTCK.
