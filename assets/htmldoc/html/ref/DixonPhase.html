DixonPhase

DixonPhase[real,imag,echos]

calculates the b0 and ph0 maps.

Details

  • The following options can be given:
  • DixonBipolarTrueDixonBipolar is an option for DixonReconstruct. If set to true it assumes alternating readout directions.
    DixonPrecessions-1DixonPrecessions is an options for DixonReconstruct. Defines the rotation of the signal {-1,1} default is -1.
    DixonFieldStrength3DixonFieldStrength is an options for DixonReconstruct. Defines the fieldstrengths in Tesla on which the data was acquired.
    DixonNucleus"1H"DixonNucleus is an option for DixonReconstruct. Defines the nucleus for which the reconstruction is performed.
    DixonFrequencies{{0}, {3.8, 3.4, 3.1, 2.7, 2.5, 1.95, 0.5, -0.5, -0.6}}DixonFrequencies is an options for DixonReconstruct. Defines the frequencies in ppm of the fat peaks being used.
    DixonAmplitudes{{1}, {0.088, 0.628, 0.059, 0.064, 0.059, 0.01, 0.039, 0.01, 0.042}}DixonAmplitudes is an options for DixonReconstruct. Defines the relative amplitudes of the fat peaks being used.
    MonitorCalcFalseMonitorCalc is an option for many processing functions. When true the proceses of the calculation is shown.
    UnwrapDimension"3D"UnwrapDimension is an option for Unwrap. Can be "2D" or "3D". 2D is for unwarpping 2D images or unwrapping the individual images from a 3D dataset (does not unwrap in the slice direction). 3D unwraps a 3D dataset in all dimensions.
    MaxIterations25MaxIterations is an option that specifies the maximum number of iterations that should be tried in various built-in functions and algorithms.
