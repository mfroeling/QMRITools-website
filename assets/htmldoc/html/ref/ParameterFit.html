ParameterFit

ParameterFit[data] fits a (skew)Normal probability density function to the data. ParameterFit[{data1, data2,...}] fits a (skew)Normal probability density function to each of the datasets. Is used in Hist.

Details

  • The following options can be given:
  • FitFunction"SkewNormal"FitFunction is an option for ParameterFit. Options are "Normal" or "SkewNormal". Indicates which function wil be fitted.
    FitOutput"Parameters"FitOutput is an option for ParameterFit and ParameterFit2. Option can be "Parameters", "Function" or "BestFitParameters".
    MethodAutomaticMethod is an option for various algorithm-intensive functions that specifies what internal methods they should use.
