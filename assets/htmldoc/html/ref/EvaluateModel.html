EvaluateModel

EvaluateModel[points,mesh]

return a dynamic plot showing an evaluation of the shape model. The points and mesh are obtained from TemplateToVolume.

EvaluateModel[{mean,mat},points,cells]

does the same but for a explicit shape model given by mean and mat. the shape model can be obtained from MakeShapeModel.
