Mask

Mask[data]

creates a mask by automatically finding a threshold.

Mask[data,min]

creates a mask which selects only data above the min value.

Mask[data,{min,max}]

creates a mask which selects data between the min and max value.

Details

  • The following options can be given:
  • MaskSmoothingFalseMaskSmoothing is an options for Mask, SmoothMask and SmoothSegmentation, if set to True it smooths the mask, by closing holse and smoothing the contours.
    MaskComponents2MaskComponents is an option for Mask, SmoothMask and SmoothSegmentation. Determinse the amount of largest clusters used as mask.
    MaskClosingFalseMaskClosing is an option for Mask, SmoothMask and SmoothSegmentation. The size of the holes in the mask that will be closed.
    MaskFiltKernel2MaskFiltKernel is an option for Mask, SmoothMask and SmoothSegmentation. How mucht the contours are smoothed.
    MaskDilation0MaskDilation is an option for Mask, SmoothMask and SmoothSegmentation. If the value is greater than 0 it will dilate the mask, if the value is smaller than 0 it will erode the mask.
    SmoothItterations3SmoothItterations is an option for Mask, SmoothMask and SmoothSegmentation and defines how often the smoothing is repeated.
