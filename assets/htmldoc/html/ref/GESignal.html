GESignal

GESignal[ang, {tr, t1}] calculates the gradient echo signal for flipangles ang using tr and t1. GESignal[ang_?ListQ, {{tr1_, tr2_}, t1_}] calculates the dual tr gradient echo signal for flipangles ang using tr1, tr2 and t1.
