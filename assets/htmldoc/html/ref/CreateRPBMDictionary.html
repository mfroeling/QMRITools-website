CreateRPBMDictionary

CreateRPBMDictionary[tm]

creates a dictionary of RPBM functions for the given mixing times tm. Options are the used parameters in the dictionary and the simulated RD values, {pars, sim}.

Details

  • The following options can be given:
  • RPBMRangeTau{100, 2000}RPBMRangeTau is an option for CreateRPBMDictionary. Default is {100, 2000}. The range of the tau values used in the RPBM function.
    RPBMRangeZeta{0.1, 3}RPBMRangeZeta is an option for CreateRPBMDictionary. Default is {0.1, 3}. The range of the zeta values used in the RPBM function.
    RPBMDictionarySize{100, 1000}RPBMDictionarySize is an option for CreateRPBMDictionary. Default is {100, 1000}.     The number of dictionaries and number of points in the dictionary.
