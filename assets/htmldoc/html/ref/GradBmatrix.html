GradBmatrix

GradBmatrix[Gt,hw,te,t]

Calculates the true bmatrix from the sequence created by GradSeq.

Details and Options

  • The following options can be given:
  • OutputPlotFalseOutputPlot is an option for GradBmatrix. It specifies if the plots of the gradients should also be exported.
    Method"Analytical"Method is an option for various algorithm-intensive functions that specifies what internal methods they should use.
    StepSizeI0.025StepSizeI is an option for GradBmatrix. Specifies the integration stepsize is Method -> "Numerical" is used.
