RPBMFunction

RPBMFunction[tm,{d0,tau,zeta}]

generates the RPBM function for the given mixing times tm, with d0, tau and zeta.

RPBMFunction[tm,{tau,zeta}]

generates the RPBM function for the given mixing times tm, assuming a d0 of 1.

RPBMFunction[tm,{{d0,tau,zeta},..}]

generates the RPBM function for the given mixing times tm, for a list of d0, tau and zeta.
