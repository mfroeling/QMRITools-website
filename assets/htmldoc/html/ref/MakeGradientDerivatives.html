MakeGradientDerivatives

MakeGradientDerivatives[vox,type]

generates two interpolation functions using the resolution defined by vox. One of the gradient field and one of the dx dy and dz of each of the gradient componets. The type is the gradient of the system which now can be "WA1" or "WA2".

Examples
