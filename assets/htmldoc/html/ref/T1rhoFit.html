T1rhoFit

T1rhoFit[data, EchoTimes] fits the T1rho value to the data using linear or nonlinear methdos. The output is in units as defined by the EchoTimes, e.g. if EchoTimes is in ms the output is in ms. Output is {S(0), T1rhomap}.
