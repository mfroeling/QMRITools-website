HarmonicDenoiseTensor

HarmonicDenoiseTensor[tens,mask,vox]

uses the harmonic denoising method to denoise the tensor within the mask. Values for which the tensor is not defined but are within the mask will be filled in.

HarmonicDenoiseTensor[tens,seg,vox]

will do the same for each segmentation label in seg.

HarmonicDenoiseTensor[tens,seg,vox,labs]

will do the same for each segmentation label in seg that is specified in labs.

HarmonicDenoiseTensor[]

is based on 10.1016/j.media.2011.01.005.
