ReadDicomDiff

ReadDicomDiff[folder,part]

imports all dicom files from the given folder and the corresponding diffusion parameters. part is the number of diffusion images per slice including the unweighted images.

Details and Options

  • The following options can be given:
  • ScaleCorrectFalseScaleCorrect is an option for ReadDicom, ReadDicomDiff, ReadDicomDir and ReadDicomDirDiff. The dicom image values are corrected for rescale slope, scale slope and rescale intercept.
