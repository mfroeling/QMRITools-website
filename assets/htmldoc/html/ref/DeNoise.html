DeNoise

DeNoise[data,sigma,filtersize]

removes Rician noise with standard deviation "sigma" from the given dataset using a kernel with size "filtersize" a gaussian kernel.

DeNoise[data,sigma,filtersize,Kernel"kerneltype"]

removes Rician noise with standard deviation "sigma" from the given dataset using a kernel with size "filtersize" and type "kerneltype". Output is data denoised.

DeNoise[]

is based on DOI: 10.1109/TMI.2008.920609.

Details and Options

  • The following options can be given:
  • DeNoiseKernel"Gaussian"DeNoiseKernel is and option for DeNoise. Values can be "Disk", "Box" or "Gaussian".
    DeNoiseMonitorFalseDeNoiseMonitor is and option for DeNoise. Monitor the denoising progres.
    DeNoiseIterations1DeNoiseIterations is and option for DeNoise. Specifies the number of the denoising iterations.
