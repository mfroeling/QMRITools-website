ApodizeFid

ApodizeFid[fid]

performs apodization on the fid. The apodization function is set with the option ApodizationFunction.

Details

  • The following options can be given:
  • ApodizationFunction"Hanning"ApodizationFunction is an options for ApodizeFid, ApodizeSpectra, ApodizePadFid, and ApodizePadSpectra. Values can be "Hanning", "Hamming", "Gaussian", "Lorentzian", and "Voigt".
