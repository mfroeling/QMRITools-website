CalculateWallMap

CalculateWallMap[mask,vox]

calculates the wall distance map and the wall derivative. Output is {wallmap, wallDerivative}.

Details and Options

  • The following options can be given:
  • ShowPlotTrueShowPlot is an option for CentralAxes, HelixAngleCalc and CardiacCoordinateSystem. True shows the fit of the central axes and outpu the plot as extra output.
    MaskWallMapFalseMaskWallMap is an option for CalculateWallMap. if True or False.
