SegmentTracts

SegmentTracts[tracts,segs,vox,dim]

segments the tracts based on segs.

Details

  • The following options can be given:
  • FiberLengthRange{20, 500}FiberLengthRange is an option for FiberTractography and specifies the allowed tract range.
    OutputForm"Joined"OutputForm[expr] prints as a two-dimensional representation of expr using only keyboard characters.
    FitTractsTrueFitTracts[tract] fits a tract or a list of tracts, each defined as a list of {x, y, z} coordinates with a polinomial function. FitTracts[tract, vox, dim] does the same but contrains all the tract coordinates to the volume difined by dim.
