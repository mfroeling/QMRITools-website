AugmentImageData

AugmentImageData[image,{rotate,flip}]

augments the input image by rotating between -180 and 180 degrees and flipping. The inputs rotate and flip can be set to True or False.

AugmentImageData[{image,..},{rotate,flip}]

same but for a list of images.
