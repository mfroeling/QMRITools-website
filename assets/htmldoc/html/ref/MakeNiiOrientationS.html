MakeNiiOrientationS

MakeNiiOrientationS[off,vox]

maxes the srow values for nii header assuming not rot and Q.

MakeNiiOrientationS[off,vox,rot]

maxes the srow values for nii header using rotation rot.

MakeNiiOrientationS[off,vox,rot,Q]

maxes the srow values for nii header using rotation rot and skew Q.
