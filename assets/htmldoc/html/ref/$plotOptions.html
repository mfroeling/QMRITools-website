$plotOptions

$plotOptions

is a list of options for plotting.

Details

  • The following options can be given:
  • AxesFalseAxes is an option for graphics functions that specifies whether axes should be drawn.
    Frame{{True, False}, {True, False}}Frame is an option for Graphics, Grid, and other constructs that specifies whether to include a frame.
    PlotHighlightingNonePlotHighlighting is an option to Plot, ListPlot and related visualization functions that specifies how points and curves should be highlighted.
    FrameStyleDirective[Thickness[Large], GrayLevel[0]]FrameStyle is an option for Graphics, Grid, and other constructs that specifies the style in which to draw frames.
    LabelStyleDirective[GrayLevel[0], Bold, 14]LabelStyle is an option for formatting and related constructs that specifies the style to use in displaying their label-like elements.
