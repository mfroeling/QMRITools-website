MuscleBidsAnalysis

MuscleBidsAnalysis[dir]

performs analysis on the Muscle-Bids named nii based on the config file in the bids sourceFolder dir. If a segmentation is present it is used to calculate the mean per segmentation.

Details

  • The following options can be given:
  • ProcessSubjectsAllProcessSubjects is an option for Bids functions. Can be a list of bids subject names else it is All.
    VersionCheckFalseVersionCheck is an option for all Bids functions. If set True data processed with an old version is reprocessed.
    BidsOutputImages"All"BidsOutputImages is an option for MuscleBidsAnalysis. If set True the output images are saved in the output folder.
