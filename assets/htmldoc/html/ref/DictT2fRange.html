DictT2fRange

DictT2fRange is an option for CreateT2Dictionary and EPGT2Fit. is specifies the range and step of the T2 fat values in the dictionary {min, max, step} in ms. If a single value is given this fixed value is used a long as EPGCalibrate is False.
