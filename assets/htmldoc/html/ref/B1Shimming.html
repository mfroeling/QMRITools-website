B1Shimming

B1Shimming[b10, b190, mask] finds the optimal shim values to shim to 100% b1. Assumes B1Scaling "Relative". B1Shimming[b10, b190, mask, target] finds the optimal shim values to shim to target, which can be a number or a map.

Details

  • The following options can be given:
  • B1ShimMethod"All"B1ShimMethod is an option for B1Shimming. Values can be "All", "Phase" or "Magnitude".
    B1MaxPower1.5B1MaxPower is an option for B1Shimming. Specifies how much power can be used per channel.
    B1EqualPowerFalseB1EqualPower is an option for B1shimming. If true equal power for both channels is used.
    B1Scaling"Relative"B1Scaling is an option for B1Shimming and CombineB1. Values can be "Relative" or "Absolute". "Absolute" assurmes b1 maps are given in uT, "Relative" assumes that maps are in %.
