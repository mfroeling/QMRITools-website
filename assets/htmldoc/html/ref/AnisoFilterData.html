AnisoFilterData

AnisoFilterData[data]

Filter the diffusion tensor data using an anisotropic filter based on the strucure tensor of the data. Output is the smoothed data.

AnisoFilterData[]

is based on DOI: 10.1016/j.jbiomech.2021.110540 and 10.1016/j.mri.2009.10.001 and 10.1371/journal.pone.0126953.

Details

  • The following options can be given:
  • AnisoStepTime1AnisoStepTime is an option for AnisoFilterTensor and defines the diffusion time, when small more step are needed.
    AnisoItterations1AnisoItterations is an options for AnisoFilterData. It specifies the amount of denoising itterations.
    AnisoKernel{0.25, 0.5}AnisoKernel is an options for AnisoFilterData. It defines the kernel size.
