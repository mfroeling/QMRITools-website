SplitDataForSegmentation

SplitDataForSegmentation[data]

is a specific function for leg data to prepare data for segmentation. It detects the side and location and will split and label the data accordingly.

SplitDataForSegmentation[data ,seg]

does the same but is rather used when preparing training data. Here the seg is split in exactly the same way as the data.
