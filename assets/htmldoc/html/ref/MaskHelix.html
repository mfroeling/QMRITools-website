MaskHelix

MaskHelix[helix, mask] masks helix angle data, sets the background to -100 and allows for Median filter of the helix mask. helix can be a singel map or the FAM. Output is the masked helix angle data.

Details

  • The following options can be given:
  • BackgroundValue-100BackgroundValue is an option for MaskHelix. Sets the backgroud value (default is -100).
    SmoothHelixFalseSmoothHelix is an option for MaskHelix, sets the kernelsize for the MedianFilter.
