T2Fit

T2Fit[data,EchoTimes]

fits the t2 value to the data using linear or nonlinear methods.vThe output is in units as defined by the EchoTimes, e.g. if EchoTimes is in ms the output is in ms.vOutput is {S(0), t2}.
