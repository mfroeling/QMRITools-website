CombineB1

CombineB1[b10,b190,{f1,f2,ang}]

combines the complex b1 maps with relative amplitudes f1 and f2 using phase angle ang.

Details and Options

  • The following options can be given:
  • B1Scaling"Relative"B1Scaling is an option for B1Shimming and CombineB1. Values can be "Relative" or "Absolute". "Absolute" assurmes b1 maps are given in uT, "Relative" assumes that maps are in %.
