ElastixTools

QMRITools package: Functions for data registration using the Elastix software (link and link).

RegisterData — Registers data to a given target.

RegisterCardiacData — Multi-slice 2D registration made for cardiac DWI data.

RegisterDiffusionData — Registration of DWI data with or without a anatomical target.

RegisterTensorData — Registration of a tensor based on the xx, yy and zz components.

$debugElastix — Toggle for Elastix interface debugging.
