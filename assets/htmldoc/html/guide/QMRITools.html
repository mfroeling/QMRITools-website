QMRITools

QMRITools package written by Martijn Froeling.  Contains various sub-packages with functions for processing, evaluating, simulating, importing, exporting and simulating of quantitative MRI data. It provides a range of functions to process DWI, DTI, Dixon and T2mapping data, phase unwrapping etc. If additional features are required please contact me on m.froeling@gmail.com

CardiacTools — Functions for processing cardiac MRI data.

DenoiseTools — Functions for noise suppression of multi dimensional MRI data.

DixonTools — Functions for Dixon data reconstruction and analysis.

ElastixTools — Functions for data registration using the Elastix software.

GeneralTools — Functions that are used in other tools.

GradientTools — Functions for diffusion gradient optimization and manipulation.

IVIMTools — Functions for fitting and analysis of IVIM MRI data.

JcouplingTools — Functions for simulation of j-coupled spectra.

MaskingTools — Functions for generating and processing of masks and segmentations.

NiftiTools — Functions for importing and exporting Nifti (*.nii) data.

PlottingTools — Functions for displaying and visualization of multi dimensional data.

ProcessingTools — Functions for generals processing of data.

ReconstructionTools — Functions for basic MRI reconstruction and coil combination.

RelaxometryTools — Functions for fitting and analysis of relaxometry data (e.g. T2, T1 and T1rho).

SimulationTools — Functions for simulating various MRI contrasts such as DWI, DTI, Dixon and EPG data.

SpectroTools — Functions for fitting, processing and analysis of MRS data.

TaggingTools — Functions for the analysis of tagging MRI data.

TensorTools — Functions for fitting processing and analysis of DTI data.

TractographyTools — Functions for DTI based fiber tractography.
